# Parscal's Triangle

## 1. 이해
    [
         [1],
        [1,1],
       [1,2,1],
      [1,3,3,1],
     [1,4,6,4,1]
    ]
    행의 수 numRow를 입력하면 위와 같은 규칙의 파스칼의 삼각형을 배열로 출력하라.

## 2. 계획
    1. 행 반복문을 돌림
    2. 행 수와 같은 수의 열 반복문을 돌림
    3. 양 끝열은 1, 가운데는 (윗행의 자리수-1에 있는 수) + (윗행의 자리수에 있는 수)
