# 두 정수 사이의 합

## 1. 이해
    두 정수 a, b가 주어졌을 때 a와 b 사이에 있는 모든 정수의 합을 구한다.

## 2. 계획
    1. a, b 값을 입력받음
    2. 입력받은 a와 b 사이의 값을 loop로 뽑아내기
    3. loop에서 나온 값을 모두 더한 값을 결과값으로 출력
