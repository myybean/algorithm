# 같은 숫자는 싫어

## 1. 이해
    숫자의 배열 중 연속적으로 나타나는 숫자는 그 연속의 최초만 남기고 제거한다.

## 2. 계획
    1. 배열의 원소를 loop로 돌려 한개씩 꺼냄
    2. ex의 값이 없는 경우 answer배열에 넣음
    3. (ex의 값이 있는 경우) ex와 현재 값을 비교해 같지 않은 경우에만 현재값을 answer배열에 넣음
    4. answer 배열을 리턴
